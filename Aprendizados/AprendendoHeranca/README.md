# 🧬 Aprendendo Herança e Polimorfismo em Java (UTFPR)

Pasta dedicada ao estudo dos pilares de Reuso e Flexibilidade na Orientação a Objetos.

## 🎯 Objetivo
Compreender como estruturar hierarquias de classes para evitar duplicação de código e permitir que o sistema trate diferentes objetos de forma genérica.

## 📂 Conteúdo e Exercícios
1.  **Conceitos de Herança:** Uso da palavra-chave `extends` para criar sub-classes.
2.  **Modificadores de Acesso:** Diferença prática entre `private`, `public` e `protected`.
3.  **Sobrescrita de Métodos (@Override):** Redefinição de comportamentos específicos em classes filhas.
4.  **Construtores na Herança:** Utilização do método `super()` para inicializar a superclasse.
5.  **Laboratório ClubChess:** Protótipo de sistema de xadrez utilizando uma classe mãe `Peca` para derivar movimentos específicos.

## 🧠 Conceitos-Chave
- **Superclasse vs. Subclasse:** Relação "É-UM" (Ex: `Bispo` é uma `Peca`).
- **Polimorfismo Dinâmico:** Capacidade de uma variável de tipo genérico referenciar objetos específicos.
- **Acoplamento e Coesão:** Como a herança bem aplicada melhora a arquitetura do software.
