# 📑 Lista de Exercícios 5 - Gestão de Coleções e Regras de Negócio

Este diretório contém as resoluções da quinta lista de exercícios da disciplina **Orientação a Objetos 1** do curso de ADS-UTFPR (Câmpus Ponta Grossa).

## 🎯 Foco Técnico

* **Manipulação de Coleções (ArrayList):** Gerenciamento de múltiplos objetos dentro de uma classe coordenadora (ex: Banco gerindo várias Contas).
* **Algoritmos de Busca:** Localização de objetos específicos em listas através de identificadores únicos (RA, Número da Conta, Código ou Placa).
* **Lógica de Negócio e Cálculos:** Implementação de regras para multas por atraso, controle de estoque e tarifação de estacionamento.
* **Interação entre Classes:** Reforço do conceito de **Composição** e exibição de dados via acesso direto aos atributos (sem uso do método `toString`).

## 📂 Exercícios

1. **Sistema Bancário:** Gestão de contas e titulares com operações de saque e depósito validados.
2. **Biblioteca UTFPR:** Controle de empréstimos com cálculo automático de multa por atraso (R$ 1,50/dia).
3. **Loja de Produtos:** Gestão de estoque com lógica de cadastro inteligente e fluxo de vendas.
4. **Gestão Acadêmica:** Matrícula de alunos e lançamento de múltiplas notas com cálculo de média e status de aprovação.
5. **Estacionamento:** Controle de vagas e tarifação de permanência (taxa fixa + adicional por hora).
