# 📚 Aprendendo Listas em Java (UTFPR)

Repositório dedicado ao estudo e prática de manipulação de coleções (`List` e `ArrayList`) na disciplina de Orientação a Objetos.

## 🎯 Objetivo
Dominar a lógica de armazenamento, busca e processamento de dados dinâmicos, focando na interação entre múltiplas classes (Composição).

## 📂 Exercícios Contidos
O projeto aborda 5 cenários práticos de lógica de negócios:

1.  **Gestão de Estoque:** Cadastro de produtos com verificação de disponibilidade.
2.  **Sistema Acadêmico:** Listagem de alunos e cálculo de médias por turma.
3.  **Controle de Funcionários:** Cálculo de folha de pagamento e busca por cargo.
4.  **Biblioteca Digital:** Empréstimos de livros com controle de status.
5.  **UTF-Stream:** Sistema de streaming com validação de limites por plano de assinatura (Envolvendo 3 listas e associações).

## 🧠 Conceitos Aplicados
- **Navegação de Objetos:** `objeto.getLista().getAtributo()`.
- **Iteração:** Uso de `for-each` para busca e soma (acumuladores).
- **Validação de Referência:** Tratamento de `null` após métodos de busca.
- **Encapsulamento:** Atributos privados e métodos de conveniência.

## 🛠️ Tecnologias
- Java JDK 17+
- Eclipse IDE
